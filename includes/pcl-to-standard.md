> [!IMPORTANT]
> 我们强烈建议不要在新的应用程序开发中使用可移植类库，因为它们只针对一种非常特定的 .NET 实现子集。 如果要生成可重用的库以在 .NET Framework 和其他 .NET 实现之间共享代码，建议的替换是 .NET Standard 2.0 库。 否则，请使用 .NET 5 或更高版本。 有关详细信息，请参阅 [.NET Standard](~/docs/standard/net-standard.md)。
