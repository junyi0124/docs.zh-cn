* [入门](get-started.md)：演示如何生成和运行 .NET for Apache Spark 应用程序。
* [将 .NET for Apache Spark 应用程序部署到 Azure HDInsight](hdinsight-deployment.md)：演示如何将 .NET for Apache Spark 应用程序部署到 Azure HDInsight。
* [将 .NET for Apache Spark 应用程序部署到 Amazon EMR Spark](amazon-emr-spark-deployment.md)：演示如何将 .NET for Apache Spark 应用程序部署到 Amazon EMR Spark。
* [将 a .NET for Apache Spark 应用程序部署到 Databricks](databricks-deployment.md)：演示如何将 .NET for Apache Spark 应用程序部署到 Databricks。
