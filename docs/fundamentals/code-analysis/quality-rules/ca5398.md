# <a name="ca5398-avoid-hardcoded-sslprotocols-values"></a>CA5398：避免硬编码的 SslProtocols 值

| | 值 |
|-|-|
| **规则 ID** |CA5398|
| **类别** |Microsoft.Security|
| **修复是中断或非中断** |不间断|

## <a name="cause"></a>原因

如果满足以下任一条件，则会触发此规则：

- 引用了安全但硬编码的 <xref:System.Security.Authentication.SslProtocols?displayProperty=nameWithType> 值。
- 表示安全协议版本的整数值被分配给 <xref:System.Security.Authentication.SslProtocols> 变量，用作  <xref:System.Security.Authentication.SslProtocols> 返回值，或用作 <xref:System.Security.Authentication.SslProtocols> 参数。

Safe 值 ar：

- Tls12
- Tls13

## <a name="rule-description"></a>规则说明

传输层安全 (TLS) 保护计算机之间的通信，最常见的是通过超文本传输协议安全 (HTTPS) 。 协议 1.1 1.0 版本不推荐使用，而 TLS 1.2 和 TLS 1.3 是最新的。 未来，TLS 1.2 和 TLS 1.3 可能已弃用。 若要确保应用程序的安全性，请避免硬编码协议版本。 有关详细信息，请参阅 [传输层安全性 (TLS) 最佳实践与 .NET Framework](../../../framework/network-programming/tls.md)。

## <a name="how-to-fix-violations"></a>如何解决冲突

不要硬编码 TLS 协议版本。

## <a name="when-to-suppress-warnings"></a>何时禁止显示警告

需要连接到无法升级的旧服务，以使用将来的 TLS 协议版本。

## <a name="pseudo-code-examples"></a>伪代码示例

### <a name="enumeration-name-violation"></a>枚举名称冲突

```csharp
using System;
using System.Security.Authentication;

public class ExampleClass
{
    public void ExampleMethod()
    {
        // CA5398 violation
        SslProtocols sslProtocols = SslProtocols.Tls12;
    }
}
```

```vb
Imports System
Imports System.Security.Authentication

Public Class TestClass
    Public Function ExampleMethod() As SslProtocols
        ' CA5398 violation
        Return SslProtocols.Tls12
    End Function
End Class
```

### <a name="integer-value-violation"></a>整数值冲突

```csharp
using System;
using System.Security.Authentication;

public class ExampleClass
{
    public SslProtocols ExampleMethod()
    {
        // CA5398 violation
        return (SslProtocols) 3072;    // TLS 1.2
    }
}
```

```vb
Imports System
Imports System.Security.Authentication

Public Class TestClass
    Public Function ExampleMethod() As SslProtocols
        ' CA5398 violation
        Return CType(3072, SslProtocols)   ' TLS 1.2
    End Function
End Class
```

### <a name="solution"></a>解决方案

```csharp
using System;
using System.Security.Authentication;

public class TestClass
{
    public void Method()
    {
        // Let the operating system decide what TLS protocol version to use.
        // See https://docs.microsoft.com/dotnet/framework/network-programming/tls
        SslProtocols sslProtocols = SslProtocols.None;
    }
}
```

```vb
Imports System
Imports System.Security.Authentication

Public Class TestClass
    Public Sub ExampleMethod()
        ' Let the operating system decide what TLS protocol version to use.
        ' See https://docs.microsoft.com/dotnet/framework/network-programming/tls
        Dim sslProtocols As SslProtocols = SslProtocols.None
    End Sub
End Class
```
