# <a name="copying-document-fragments"></a>复制文档片段

可以先创建 XmlDocumentFragment  节点，再在它下面添加节点。 如果使用 InsertNode  方法插入 XmlDocumentFragment  ，不会复制 XmlDocumentFragment  节点，但会在 XML 文档对象模型 (DOM) 中插入它的子节点。
