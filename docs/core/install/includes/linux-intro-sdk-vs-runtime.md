如果要开发 .NET 应用，请安装 SDK（包括运行时）。 或者，如果只需运行应用程序，请安装运行时。 如果要安装该运行时，建议安装 ASP.NET Core 运行时，因为它同时包括 .NET 和 ASP.NET Core 运行时。

如果已安装 SDK 或运行时，请使用 `dotnet --list-sdks` 和 `dotnet --list-runtimes` 命令查看安装了哪些版本。 有关详细信息，请参阅[如何检查是否已安装 .NET](../how-to-detect-installed-versions.md)。
