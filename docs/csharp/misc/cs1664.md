# <a name="compiler-error-cs1664"></a>编译器错误 CS1664

长度为“length”、类型为“type”的固定大小缓冲区太大  
  
 固定大小缓冲区的最大大小（由长度乘以元素大小决定）为 2^31 = 268435455。
