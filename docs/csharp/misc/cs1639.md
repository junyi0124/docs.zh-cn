# <a name="compiler-error-cs1639"></a>编译器错误 CS1639

接口“interface”的托管组件类包装器类签名“signature”不是有效的类名签名  
  
 如果该接口的元数据无效，将发生此错误。 如果使用 C# 或受支持的 .NET 语言生成接口，则该错误永远不会发生。 与该接口的供应商一起进行检查，或验证已成功生成的接口程序集。
