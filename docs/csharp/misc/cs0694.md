# <a name="compiler-error-cs0694"></a>编译器错误 CS0694

类型形参“identifier”与包含类型或方法同名  
  
 必须为类型形参使用不同名称，因为类型形参不能与包含该类型形参的类型或方法同名。  
  
## <a name="example"></a>示例  

 以下示例生成 CS0694。  
  
```csharp  
// CS0694.cs  
// compile with: /target:library  
class C<C> {}   // CS0694  
```  
  
## <a name="example"></a>示例  

 除了涉及泛型类的以上示例，使用一种方法也可能发生此错误：  
  
```csharp  
// CS0694_2.cs  
// compile with: /target:library  
class A  
{  
   public void F<F>(F arg);   // CS0694  
}  
```
