# <a name="compiler-error-cs1022"></a>编译器错误 CS1022

应输入类型、命名空间定义或文件尾  
  
 源代码文件不具有一组匹配的大括号。  
  
 下面的示例生成 CS1022：  
  
```csharp  
// CS1022.cs  
namespace x  
{  
}  
}   // CS1022  
```
