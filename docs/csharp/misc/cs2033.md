# <a name="compiler-error-cs2033"></a>编译器错误 CS2033

包含短文件名“filename”的长文件名已存在，无法创建同名短文件名  
  
 编译名称超过八个字符的任何 C# 文件。 然后，编译另一个具有前面的文件名称的较短形式的文件，例如名称的前六个字符再加上"~1"。 第二个编译将生成此错误。  
  
 若要解决此错误，请将较短文件名重命名为一个与长文件名不冲突的文件名。
