# <a name="compiler-error-cs0228"></a>编译器错误 CS0228

“type”不包含“member”的定义，或者它是不可访问的  
  
 可能已经替换了 **System.Hashtable**、 **System.String**或 **System.Array**的系统版本，并且此替换不包含 `member`。  
  
 否则，你可能需要修复或重新安装 Visual Studio。
