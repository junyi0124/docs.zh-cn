# <a name="compiler-error-cs1514"></a>编译器错误 CS1514

应为 {  
  
 找不到编译器需要的左大括号 (`{`)。  
  
 下面的示例生成 CS1514：  
  
```csharp  
// CS1514.cs  
namespace x  
// CS1514, no open curly brace  
```
