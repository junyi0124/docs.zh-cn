# <a name="compiler-error-cs0152"></a>编译器错误 CS0152

标签“label”已经出现在该 switch 语句中  
  
 标签在 [switch](../language-reference/keywords/switch.md) 语句中重复。 有关更多信息，请参见 [switch](../language-reference/keywords/switch.md)。  
  
 以下示例生成 CS0152：  
  
```csharp  
// CS0152.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
  
         switch (i)  
         {  
            case 1:  
               i++;  
               return;  
  
            case 1:   // CS0152, two case 1 statements  
               i++;  
               return;  
         }  
      }  
   }  
}  
```
