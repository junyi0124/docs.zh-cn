# <a name="compiler-error-cs1577"></a>编译器错误 CS1577

程序集生成失败 - 原因  
  
 在 C# 编译器成功编译此源代码之后，此编译的程序集生成部分失败。 请参阅 ALink 实用程序的错误文档（[Al.exe 工具错误和警告](../../framework/tools/al-exe-assembly-linker.md#errors-and-warnings)），了解详细信息。
