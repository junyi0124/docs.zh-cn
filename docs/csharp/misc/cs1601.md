# <a name="compiler-error-cs1601"></a>编译器错误 CS1601

方法或委托参数不能是“type”类型  
  
 .NET 类库中的某些类型（例如、和） <xref:System.TypedReference> <xref:System.RuntimeArgumentHandle> <xref:System.ArgIterator> 不能用作 [in](../language-reference/keywords/in-parameter-modifier.md)、 [ref](../language-reference/keywords/ref.md) 或 [out](../language-reference/keywords/out-parameter-modifier.md) 参数，因为它们可能用于执行不安全的操作。  
  
 下面的示例生成 CS1601：  
  
```csharp  
// CS1601.cs  
using System;

class MyClass
{
    public void Test1(in TypedReference t)   // CS1601  
    {
    }

    public void Test2(ref TypedReference t)   // CS1601  
    {
    }

    public void Test3(out ArgIterator t)   // CS1601  
    {
        t = default;
    }
}
```
