# <a name="compiler-error-cs2016"></a>编译器错误 CS2016

代码页“codepage”无效或未安装  
  
 [-代码页](../language-reference/compiler-options/codepage-compiler-option.md)编译器选项传递了无效的值。  
  
 下面的示例生成 CS2016:  
  
```csharp  
// CS2016.cs  
// compile with: /codepage:x  
// CS2016 expected  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
