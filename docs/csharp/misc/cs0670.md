# <a name="compiler-error-cs0670"></a>编译器错误 CS0670

字段不能有 void 类型  
  
 将字段声明为类型 [void](../language-reference/builtin-types/void.md)。  
  
 以下示例生成 CS0670：  
  
```csharp  
// CS0670.cs  
class C  
{  
   void f;   // CS0670  
   // try the following line instead  
   // public int f;  
  
   public static void Main()  
   {  
      C myc = new C();  
      myc.f = 0;  
   }  
}  
```
