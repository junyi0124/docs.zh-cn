# <a name="compiler-error-cs0535"></a>编译器错误 CS0535

“class”不实现接口成员“member”  
  
 [类](../language-reference/keywords/class.md) 派生自 [接口](../language-reference/keywords/interface.md)，但该类未实现一个或多个接口的成员。 类必须实现其派生自的接口的所有成员，或者被声明为 `abstract`。  
  
## <a name="example"></a>示例  

 下面的示例生成 CS0535。  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a>示例  

 下面的示例生成 CS0535。  
  
```csharp  
// CS0535_b.cs  
using System;  
class C : IDisposable {}   // CS0535  
  
// OK  
class D : IDisposable {  
   void IDisposable.Dispose() {}  
   public void Dispose() {}  
  
   static void Main() {  
      using (D d = new D()) {}  
   }  
}  
```
