# <a name="compiler-warning-level-1-cs2014"></a>编译器警告（等级 1）CS2014

编译器选项“old option”已过时，请使用“new option”  
  
 编译器选项的格式被弃用。 有关详细信息，请参阅 [C# 编译器选项](../language-reference/compiler-options/index.md) 。
