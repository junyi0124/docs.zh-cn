# <a name="compiler-error-cs0590"></a>编译器错误 CS0590

用户定义的运算符不能返回 void  
  
 用户定义的运算符的用途是返回对象。  
  
 下面的示例生成 CS0590：  
  
```csharp  
// CS0590.cs  
namespace x  
{  
   public class a  
   {  
      public static void operator+(a A1, a A2)   // CS0590  
      {  
      }  
  
      // try the following user-defined operator  
      /*  
      public static a operator+(a A1, a A2)  
      {  
         return A2;  
      }  
      */  
  
      public static int Main()  
      {  
         return 1;  
      }  
   }  
}  
```
