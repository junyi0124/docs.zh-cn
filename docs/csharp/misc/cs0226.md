# <a name="compiler-error-cs0226"></a>编译器错误 CS0226

__arglist 表达式只能出现在调用或新的表达式内部。  
  
 不支持的关键字 `__arglist` 只能出现在方法调用或新的表达式中。  
  
## <a name="example"></a>示例  

 以下代码生成 CS0226：  
  
```csharp  
// cs0226.cs  
using System;  
  
public class C  
    {  
    public static int Main ()  
        {  
        __arglist(1,"This is a string"); // CS0226  
        return 0;  
        }  
    }  
```
