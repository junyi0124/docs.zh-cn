# <a name="compiler-error-cs1504"></a>编译器错误 CS1504

无法打开源文件“file”（“reason”）  
  
 无法打开或使用编译器读取源文件。 该文件可能被另一个应用程序锁定，或者可能有一些其他的操作系统问题。 该消息包含编译器无法打开或读取该文件的操作系统的原因。
