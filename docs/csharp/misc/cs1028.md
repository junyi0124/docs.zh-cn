# <a name="compiler-error-cs1028"></a>编译器错误 CS1028

意外的预处理器指令  
  
 已发现一个意外的 [预处理器指令](../language-reference/preprocessor-directives/index.md) 。  
  
 例如， `#endif` 前面没有找到 `#if`。  
  
 以下示例生成 CS1028：  
  
```csharp  
// CS1028.cs  
#endif   // CS1028, no matching #if  
namespace x  
{  
   public class clx  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
