# <a name="compiler-error-cs2006"></a>编译器错误 CS2006

命令行语法错误：“option”选项缺少“text”  
  
 *option* 的语法需要额外的文本。 有关信息，请参阅 [编译器选项](../language-reference/compiler-options/index.md)。
