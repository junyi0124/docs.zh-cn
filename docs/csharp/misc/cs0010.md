# <a name="compiler-error-cs0010"></a>编译器错误 CS0010

意外致命错误 --“error”。  
  
 出现会阻止编译的完全意外的内容时，会生成此错误。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 再重新编译一个项目。 如果你收到相同错误，请尝试重新安装 Visual Studio。 如果你收到此错误，请向 Microsoft 发送错误报告。
