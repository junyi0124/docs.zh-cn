# <a name="compiler-warning-level-3-cs0105"></a>编译器警告（等级 3）CS0105

“namespace”的 using 指令以前在此命名空间中出现过  
  
 一个只应声明一次的 [命名空间](../language-reference/keywords/namespace.md)声明了多次；删除所有重复的命名空间声明。  
  
 下面的示例生成 CS0105：  
  
```csharp  
// CS0105.cs  
// compile with: /W:3  
using System;  
using System;   // CS0105  
  
public class a  
{  
   public static void Main()  
   {  
   }  
}  
```
