# <a name="compiler-error-cs2013"></a>编译器错误 CS2013

图像基数“value”无效  
  
 传递给 [-baseaddress](../language-reference/compiler-options/baseaddress-compiler-option.md) 编译器选项的值 (不是数字) 。  
  
 以下示例生成 CS2013：  
  
```csharp  
// CS2013.cs  
// compile with: /target:library /baseaddress:x  
// CS2013 expected  
class MyClass  
{  
}  
```
