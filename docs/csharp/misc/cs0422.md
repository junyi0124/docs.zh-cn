# <a name="compiler-warning-level-4-cs0422"></a>编译器警告（等级 4）CS0422

不再支持 /incremental 选项

 C # 2.0 中不支持增量编译 (**/incr** 或 **/incremental**) 。

> [!NOTE]
> Roslyn 中不再使用此编译器警告。
