# <a name="compiler-error-cs0721"></a>编译器错误 CS0721

“type”：静态类型不能用作参数  
  
 静态类型作为参数没有意义。 由于不能创建静态类型的实例，所以没有实例可以作为参数传递。  
  
 下面的示例生成 CS0721：  
  
```csharp  
// CS0721.cs  
public static class SC  
{  
}  
  
public class CMain  
{  
   public void F(SC sc)  // CS0721  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
