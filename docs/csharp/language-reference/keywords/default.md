# <a name="default-c-reference"></a>default（C# 参考）

`default` 关键字有两种用法：

- 指定 [`switch` 语句](switch.md)中的默认标签。
- 作为 [default 默认运算符或文本](../operators/default.md)生成类型的默认值。
