# <a name="2025---internalcachemetadatastop"></a>2025 - InternalCacheMetadataStop

## <a name="properties"></a>属性  
  
|||  
|-|-|  
|ID|2025|  
|关键字|WFRuntime|  
|级别|“详细”|  
|通道|Microsoft-Windows-应用程序服务器-应用程序/调试|  
  
## <a name="description"></a>描述
