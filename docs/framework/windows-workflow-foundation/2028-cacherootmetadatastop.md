# <a name="2028---cacherootmetadatastop"></a>2028 - CacheRootMetadataStop

|||  
|-|-|  
|ID|2028|  
|关键字|WFRuntime|  
|级别|“详细”|  
|通道|Microsoft-Windows-应用程序服务器-应用程序/调试|
