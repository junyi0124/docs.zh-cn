# <a name="39460---trackingvaluenotserializable"></a>39460 - TrackingValueNotSerializable

## <a name="properties"></a>属性  
  
|||  
|-|-|  
|ID|39460|  
|关键字|WFTracking|  
|Level|警告|  
|通道|Microsoft-Windows-应用程序服务器-应用程序/调试|  
  
## <a name="description"></a>描述  

 指示所提取的自变量/变量不可序列化。  
  
## <a name="message"></a>消息  

 所提取的自变量/变量“%1”不可序列化。
