# <a name="systemservicemodeltxreleaseserviceinstanceoncompletion"></a>System.ServiceModel.TxReleaseServiceInstanceOnCompletion

因 ReleaseServiceInstanceOnTransactionComplete ServiceBehaviorAttribute 设置为 true 而在事务“{0}”完成时释放了服务实例。  
  
## <a name="description"></a>描述  

 因当前活动事务已完成而释放当前服务实例以及 ReleaseServiceInstanceOnTransactionComplete 设置为 `true` 时，将跟踪该服务实例。
