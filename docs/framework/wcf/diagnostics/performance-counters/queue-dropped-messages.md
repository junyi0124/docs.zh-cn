# <a name="queue-dropped-messages"></a>Queue Dropped Messages（队列丢弃的消息数）

计数器名称：Queued Messages Dropped（已丢弃的排队消息数）。  
  
## <a name="description"></a>描述  

 此服务中由排队传输丢弃的消息数。  
  
 有关何时丢弃消息的详细信息，请参阅 [病毒消息处理](../../feature-details/poison-message-handling.md)。
