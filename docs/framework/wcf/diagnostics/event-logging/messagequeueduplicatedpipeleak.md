# <a name="messagequeueduplicatedpipeleak"></a>MessageQueueDuplicatedPipeLeak

Id：166  
  
 严重性：错误  
  
 类别：SMSvcHost  
  
## <a name="description"></a>描述  

 此事件指示调度重复的命名管道时出错。 此句柄当前已在进程中泄漏。 此事件将列出源、异常、进程名称和进程 ID。
