# <a name="thumbprintnotvalidated"></a>ThumbPrintNotValidated

ID：153  
  
 严重性：错误  
  
 类别：TransactionBridge  
  
## <a name="description"></a>描述  

 此事件指示无法验证带有指定指纹的标识证书。 该事件将列出进程名称和进程 ID。
