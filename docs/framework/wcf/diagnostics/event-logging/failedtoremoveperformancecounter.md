# <a name="failedtoremoveperformancecounter"></a>FailedToRemovePerformanceCounter

ID：119  
  
 严重性：错误  
  
 类别：ServiceModel  
  
## <a name="description"></a>描述  

 此事件指示未移除性能计数器。 该事件列出了类别名称、计数器名称、实例名称、异常、进程名称和进程 ID。
