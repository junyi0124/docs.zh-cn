# <a name="messageloggingoff"></a>MessageLoggingOff

Id：117  
  
 严重性：错误  
  
 类别：ServiceModel  
  
## <a name="description"></a>描述  

 此事件指示消息日志记录已关闭。 该事件将列出进程名称和进程 ID。
