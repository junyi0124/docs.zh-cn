# <a name="wmiadmintypemismatch"></a>WmiAdminTypeMismatch

ID：128  
  
 严重性：错误  
  
 类别：ServiceModel  
  
## <a name="description"></a>描述  

 此事件指示出现类型不匹配。 指定的 CIM 类属性无法接受特定类型的值。 该事件将列出进程名称和进程 ID。
