# <a name="starterrorpublish"></a>StartErrorPublish

ID：158  
  
 严重性：错误  
  
 类别：SMSvcHost  
  
## <a name="description"></a>描述  

 此事件指示 NT 服务启动失败。 无法发布侦听终结点。 该事件将列出进程名称和进程 ID。
