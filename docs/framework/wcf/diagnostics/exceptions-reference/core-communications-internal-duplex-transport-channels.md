本主题列出了 Windows Communication Foundation (WCF) 内部双工传输通道生成的所有异常。  
  
## <a name="exception-list"></a>异常列表  
  
|资源代码|资源字符串|  
|-------------------|---------------------|  
|MessageMustHaveViaOrToSetForSendingOnServerSideCompositeDuplexChannels|要通过服务器复合双工通道发送消息，该消息必须设置了“Via”属性或“To”标头。|
