# <a name="multiple-active-result-sets-mars"></a>多个活动的结果集 (MARS)

多重活动结果集 (MARS) 是一项可便于在一个连接上执行多个批处理的功能。 在旧版中，一次只能对一个连接执行一个批处理。 使用 MARS 执行多个批处理并不意味着同时执行操作。
