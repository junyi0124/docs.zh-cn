# <a name="icorprofilerthreadenumgetcount-method"></a>ICorProfilerThreadEnum::GetCount 方法

获取应用程序使用的线程数。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetCount (    [out] ULONG * pcelt  
);  
```  
  
## <a name="parameters"></a>参数  

 `celt`  
 弄应用程序使用的线程数。
