# <a name="icordebugunmanagedcallback-interface"></a>ICorDebugUnmanagedCallback 接口

提供与公共语言运行时 (CLR) 不直接相关的本机事件的通知。  
  
## <a name="methods"></a>方法  
  
|方法|说明|  
|------------|-----------------|  
|[DebugEvent 方法](icordebugunmanagedcallback-debugevent-method.md)|通知调试器已激发本机事件。|  
  
## <a name="remarks"></a>注解  
  
> [!NOTE]
> 此接口不支持跨计算机或跨进程远程调用。
