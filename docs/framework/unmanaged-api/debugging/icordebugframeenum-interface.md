# <a name="icordebugframeenum-interface"></a>ICorDebugFrameEnum 接口

实现 ICorDebugEnum 方法，并枚举 ICorDebugFrame 数组。  
  
## <a name="methods"></a>方法  
  
|方法|说明|  
|------------|-----------------|  
|[Next 方法](icordebugframeenum-next-method.md)|`ICorDebugFrame`从当前位置开始，从枚举中获取指定数目的实例。|  
  
## <a name="remarks"></a>注解  
  
> [!NOTE]
> 此接口不支持跨计算机或跨进程远程调用。
