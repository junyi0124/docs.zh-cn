# <a name="icordebugappdomain4-interface"></a>ICorDebugAppDomain4 接口

对 ICorDebugAppDomain 接口进行逻辑扩展，以从 COM 可调用包装器获取托管对象。  
  
## <a name="method"></a>方法  
  
|方法|说明|  
|------------|-----------------|  
|[GetObjectForCCW 方法](icordebugappdomain4-getobjectforccw-method.md)|从 COM 可调用包装器 (CCW) 指针获取托管对象。|
