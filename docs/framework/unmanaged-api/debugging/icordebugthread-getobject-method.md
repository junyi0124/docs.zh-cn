# <a name="icordebugthreadgetobject-method"></a>ICorDebugThread::GetObject 方法

 (CLR) 线程获取公共语言运行时的接口指针。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetObject (  
    [out] ICorDebugValue   **ppObject  
);  
```  
  
## <a name="parameters"></a>参数  

 `ppObject`  
 弄指向表示 CLR 线程的 ICorDebugValue 接口对象地址的指针。
