# <a name="logswitchcallreason-enumeration"></a>LogSwitchCallReason 枚举

指示已对调试/跟踪开关执行的操作。  
  
## <a name="syntax"></a>语法  
  
```cpp  
typedef enum LogSwitchCallReason {  
    SWITCH_CREATE,  
    SWITCH_MODIFY,  
    SWITCH_DELETE  
} LogSwitchCallReason;  
```  
  
## <a name="members"></a>成员  
  
|成员|说明|  
|------------|-----------------|  
|`SWITCH_CREATE`|已创建调试/跟踪开关。|  
|`SWITCH_MODIFY`|已修改调试/跟踪开关。|  
|`SWITCH_DELETE`|已删除调试/跟踪开关。|
