# <a name="icordebugclassgetmodule-method"></a>ICorDebugClass::GetModule 方法

获取定义此类的模块。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetModule (  
    [out] ICorDebugModule    **pModule  
);  
```  
  
## <a name="parameters"></a>参数  

 `pModule`  
 弄指向 ICorDebugModule 对象的地址的指针，该对象表示在其中定义此类的模块。
