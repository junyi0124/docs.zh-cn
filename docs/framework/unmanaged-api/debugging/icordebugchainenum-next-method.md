# <a name="icordebugchainenumnext-method"></a>ICorDebugChainEnum::Next 方法

从当前位置开始，从枚举中获取指定数量的 ICorDebugChain 实例。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT Next (  
    [in] ULONG  celt,  
    [out, size_is(celt), length_is(*pceltFetched)]  
        ICorDebugChain *chains[],  
    [out] ULONG *pceltFetched  
);  
```  
  
## <a name="parameters"></a>参数  

 `celt`  
 中 `ICorDebugChain` 要检索的实例数。  
  
 `chains`  
 弄一个指针数组，其中每个指针指向一个 `ICorDebugChain` 表示链的对象。  
  
 `pceltFetched`  
 弄一个指针，指向 `ICorDebugChain` 实际返回的实例数。 如果为1，则此值可以为 null `celt` 。
