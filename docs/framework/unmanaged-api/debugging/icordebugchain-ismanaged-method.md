# <a name="icordebugchainismanaged-method"></a>ICorDebugChain::IsManaged 方法

获取一个值，该值指示此链是否正在运行托管代码。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT IsManaged (  
    [out] BOOL               *pManaged  
);  
```  
  
## <a name="parameters"></a>参数  

 `pManaged`  
 [out] `true` 如果此链正在运行托管代码，则为;否则为 `false` 。
