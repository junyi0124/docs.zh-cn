# <a name="icordebugboxvaluegetobject-method"></a>ICorDebugBoxValue::GetObject 方法

获取装箱的值。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetObject (  
    [out] ICorDebugObjectValue **ppObject  
);  
```  
  
## <a name="parameters"></a>参数  

 `ppObject`  
 弄指向表示装箱值的 ICorDebugObjectValue 对象地址的指针。
