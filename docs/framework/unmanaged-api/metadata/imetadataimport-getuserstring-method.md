# <a name="imetadataimportgetuserstring-method"></a>IMetaDataImport::GetUserString 方法

获取指定元数据标记所表示的文字字符串。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetUserString (  
   [in]   mdString    stk,  
   [out]  LPWSTR      szString,  
   [in]   ULONG       cchString,  
   [out]  ULONG       *pchString  
);  
```  
  
## <a name="parameters"></a>参数  

 `stk`  
 中要为其返回关联字符串的字符串标记。  
  
 `szString`  
 弄请求的字符串的副本。  
  
 `cchString`  
 中请求的的最大大小（以宽字符为大小） `szString` 。  
  
 `pchString`  
 弄返回的的大小（以宽字符为大小） `szString` 。
