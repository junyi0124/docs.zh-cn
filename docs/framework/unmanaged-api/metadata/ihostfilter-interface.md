# <a name="ihostfilter-interface"></a>IHostFilter 接口

提供一个方法，用于指示将处理指定的标记。  
  
> [!NOTE]
> `IHostFilter`接口必须由客户端实现。  
  
## <a name="methods"></a>方法  
  
|方法|说明|  
|------------|-----------------|  
|[MarkToken 方法](ihostfilter-marktoken-method.md)|指示将处理指定的元数据标记。|
