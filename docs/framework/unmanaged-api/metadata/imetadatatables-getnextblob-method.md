# <a name="imetadatatablesgetnextblob-method"></a>IMetaDataTables::GetNextBlob 方法

获取表中 (BLOB) 的下一个二进制大型对象的索引。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetNextBlob (  
    [in]  ULONG   ixBlob,  
    [out] ULONG   *pNext  
);  
```  
  
## <a name="parameters"></a>参数  

 `ixBlob`  
 中从 Blob 的列中返回的索引。  
  
 `pNext`  
 弄指向下一个 BLOB 的索引的指针。
