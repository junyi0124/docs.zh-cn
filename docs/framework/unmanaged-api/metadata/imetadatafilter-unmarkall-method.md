# <a name="imetadatafilterunmarkall-method"></a>IMetaDataFilter::UnmarkAll 方法

从当前元数据范围内的所有标记中删除处理标记。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT UnmarkAll ();  
```  
  
## <a name="parameters"></a>参数  

 `UnmarkAll` 不采用任何参数。
