# <a name="iceegengetmethodbuffer-method"></a>ICeeGen::GetMethodBuffer 方法

获取指定的相对虚拟地址处的方法的适当大小的缓冲区。  
  
 此方法已过时，不应使用。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetMethodBuffer (  
    [in]  ULONG       RVA,  
    [out] UCHAR       **lpBuffer  
);  
```  
  
## <a name="parameters"></a>参数  

 `RVA`  
 中要为其返回缓冲区的方法的相对虚拟地址。  
  
 `lpBuffer`  
 弄指向返回的缓冲区的指针。
