# <a name="imetadataimportgetnativecallconvfromsig-method"></a>IMetaDataImport::GetNativeCallConvFromSig 方法

获取指定的签名指针所表示的方法的本机调用约定。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetNativeCallConvFromSig (  
   [in]  void const  *pvSig,  
   [in]  ULONG       cbSig,  
   [out] ULONG       *pCallConv  
);  
```  
  
## <a name="parameters"></a>参数  

 `pvSig`  
 中指向要为其返回调用约定的方法的元数据签名的指针。  
  
 `cbSig`  
 中的大小（以字节为单位） `pvSig` 。  
  
 `pCallConv`  
 弄指向本机调用约定的指针。
