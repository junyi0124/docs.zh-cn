# <a name="iceegenallocatemethodbuffer-method"></a>ICeeGen::AllocateMethodBuffer 方法

为方法创建指定大小的缓冲区，并获取该方法的相对虚拟地址。  
  
 此方法已过时，不应使用。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT AllocateMethodBuffer (
    [in]  ULONG    cchBuffer,
    [out] UCHAR    **lpBuffer,  
    [out] ULONG    *RVA  
);  
```  
  
## <a name="parameters"></a>参数  

 `cchBuffer`  
 中要创建的缓冲区的长度。  
  
 `lpBuffer`  
 弄返回的缓冲区。  
  
 `RVA`  
 弄方法的相对虚拟地址。
