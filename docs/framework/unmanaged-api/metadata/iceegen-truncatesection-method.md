# <a name="iceegentruncatesection-method"></a>ICeeGen::TruncateSection 方法

按指定长度截断指定的代码段。  
  
 此方法已过时，不应使用。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT TruncateSection (  
    [in]  HCEESECTION     section,  
    [in]  ULONG           len  
);  
```  
  
## <a name="parameters"></a>参数  

 `section`  
 中要截断的部分。  
  
 `len`  
 中用于截断部分的长度（以字节为单位）。  
  
## <a name="remarks"></a>注解  

 `TruncateSection`仅当有特殊部分的要求不是由其他方法处理时才调用。
