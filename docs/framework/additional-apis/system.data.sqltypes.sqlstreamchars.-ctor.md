# <a name="sqlstreamchars-constructor"></a>SqlStreamChars 构造函数

初始化 `SqlStreamChars` 类的新实例。 包含此构造函数的程序集与 SQLAccess 具有友元关系。 它旨在 SQL Server 使用。 对于其他数据库，请使用该数据库提供的托管机制。

```csharp
protected SqlStreamChars ();
```

## <a name="remarks"></a>备注

> [!WARNING]
> @No__t 的构造函数是受保护的，不应在代码中直接使用。
>
> 在任何情况下，Microsoft 不支持在生产应用程序中使用此构造函数。
