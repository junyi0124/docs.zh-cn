# <a name="special-characters-visual-basic-reference"></a>特殊字符（Visual Basic 引用）

特殊字符是预定义的上下文字符，用于修改它们所预置的程序元素。 Visual Basic 支持以下特殊字符：

- [$](interpolated.md)，内插的字符串字符。
