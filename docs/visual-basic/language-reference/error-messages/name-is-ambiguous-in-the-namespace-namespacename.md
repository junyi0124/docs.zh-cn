# <a name="bc30560-name-is-ambiguous-in-the-namespace-namespacename"></a>BC30560： "" \<name> 在命名空间 "" 中不明确 \<namespacename>

你提供的名称不明确，因此与另一个名称冲突。 Visual Basic 编译器没有任何冲突解决规则;你必须自己区分名称。

 **错误 ID：** BC30560

## <a name="to-correct-this-error"></a>更正此错误

- 完全限定名称。
