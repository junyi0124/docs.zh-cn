# <a name="device-io-error"></a>设备 I/O 错误

当您的程序使用的是打印机或磁盘驱动器等设备时出现输入或输出错误。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 请确保设备正常运行，然后重试该操作。
