# <a name="property-or-method-not-found"></a>未找到属性或方法

引用的对象方法或对象属性未定义。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 可能拼错了对象的名称。 若要查看为对象定义的属性和方法，请显示对象浏览器。 选择相应的对象库以查看可用属性和方法的列表。
