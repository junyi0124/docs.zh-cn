# <a name="bc2004-out-of-memory"></a>BC2004：内存不足

所需内存多于可用内存。

 **错误 ID：** BC2004

## <a name="to-correct-this-error"></a>更正此错误

- 关闭不必要的应用程序、文档和源文件。

- 消除不必要的控件和窗体，以便一次加载更少的控件和窗体

- 减少变量的数量 `Public` 。

- 检查可用磁盘空间。

- 通过安装额外的内存或重新分配内存增加可用 RAM。

- 请确保在不再需要内存时释放内存。
