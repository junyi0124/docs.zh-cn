# <a name="bc30203-identifier-expected"></a>BC30203：应为标识符

如果上下文需要元素名称，则会出现一个编程元素，该元素不是可识别的声明元素名称。 一个可能的原因是在语句的开头除之外指定了某一特性。

 **错误 ID：** BC30203

## <a name="to-correct-this-error"></a>更正此错误

- 验证语句中的所有属性都位于开头。

- 验证语句中的所有元素名称拼写正确。
