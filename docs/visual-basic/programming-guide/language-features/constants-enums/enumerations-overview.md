# <a name="enumerations-overview-visual-basic"></a>枚举概述 (Visual Basic)

枚举提供了一种简便的方法来处理相关常量集，并将常量值与名称相关联。 例如，可以为一组与星期几相关联的整数常量声明一个枚举，然后在代码中使用星期几的名称而不是整数值。
