# <a name="specifier-is-not-valid-on-an-enum-declaration"></a>" \<specifier> " 在枚举声明中无效

你使用了在 `Enum` 声明上无效的说明符。  
  
 **错误 ID：** BC30396  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 删除说明符。
