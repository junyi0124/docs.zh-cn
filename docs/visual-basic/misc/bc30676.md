# <a name="eventname-is-not-an-event-of-containername"></a>" \<eventname> " 不是 "" 的事件 \<containername>

指定的事件未对此对象声明。  
  
 **错误 ID：** BC30676  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 检查事件的名称的拼写。  
  
2. 检查你是否正在访问正确对象。 使用对对象完全限定的引用，或者使用 `Imports` 语句导入相应的命名空间（如有必要）。
