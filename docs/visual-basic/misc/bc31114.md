# <a name="custom-event-must-end-with-a-matching-end-event"></a>“Custom Event”必须以匹配的“End Event”结束

`Custom Event` 声明必须以 `End Event` 语句结束。  
  
 **错误 ID：** BC31114  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 检查 `Custom Event` 声明是否以 `End Event` 语句结束。
