# <a name="before-and-after-arguments-cannot-be-combined"></a>不能组合“Before”和“After”自变量

参数列表同时包含相互排斥的 `Before` 和 `After` 参数。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 删除 `Before` 或 `After` 参数。
