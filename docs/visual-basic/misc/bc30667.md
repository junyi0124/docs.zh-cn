# <a name="paramarray-parameters-must-be-declared-byval"></a>ParamArray 参数必须声明为“ByVal”

`ParamArray` 参数不能使用 `ByRef` 修饰符。  
  
 **错误 ID：** BC30667  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 使用 `ParamArray` 修饰符声明 `ByVal` 参数。
