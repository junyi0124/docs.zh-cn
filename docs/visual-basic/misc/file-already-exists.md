# <a name="file-already-exists"></a>文件已存在

以现有文件的名称命名新文件或使用 **另存为** 命令以已存在的名称保存当前加载的项目时，会出现此错误。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 检查名称并将其更改为不冲突的名称，或者删除先前存在的文件或项目（如果不再需要）。
