# <a name="end-namespace-must-be-preceded-by-a-matching-namespace"></a>“End Namespace”前面必须是匹配的“Namespace”

命名空间块以 `Namespace` 语句开头且以 `End Namespace` 构造结尾。  
  
 **错误 ID：** BC30623  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 确保命名空间块以 `Namespace` 语句开头。
