# <a name="shared-cannot-be-combined-with-specifier-on-a-property-declaration"></a>“Shared”不能与属性声明上的“\<specifier>”组合

你试图将 `Shared` 修饰符与属性声明上的此类组合中无效的说明符组合在一起。  
  
 **错误 ID：** BC30502  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 删除说明符。
