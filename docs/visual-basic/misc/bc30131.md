# <a name="security-attribute-attributename-cannot-be-applied-to-a-module"></a>安全特性“\<attributename>”不能应用于模块

公共语言运行时不接受模块中的安全特性。

**错误 ID：** BC30131

## <a name="to-correct-this-error"></a>更正此错误

- 从声明中删除特性。
