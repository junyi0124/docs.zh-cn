# <a name="option-infer-can-be-followed-only-by-on-or-off"></a>“Option Infer”后面只能跟“On”或“Off”

`Option Infer` 语句必须指定 `On` 或 `Off`。 其他值均无效。  
  
 **错误 ID：** BC30979  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 指定 `Option Infer On` 或 `Option Infer Off`。
