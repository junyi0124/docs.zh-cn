# <a name="option-optionname-requires-argument"></a>选项 \<optionname> 需要 \<argument>

省略了必需的参数。  
  
 **错误 ID：** BC2006  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 提供缺少的参数。
