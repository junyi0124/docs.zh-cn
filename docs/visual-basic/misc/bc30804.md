# <a name="variant-is-no-longer-a-supported-type-use-the-object-type-instead"></a>“Variant”不再是受支持的类型；请改用“Object”类型

`Object` 数据类型可替换 `Variant`。  
  
 **错误 ID：** BC30804  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 将项声明为 `Object`。
