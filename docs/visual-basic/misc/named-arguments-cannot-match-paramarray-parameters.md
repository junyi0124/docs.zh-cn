# <a name="named-arguments-cannot-match-paramarray-parameters"></a>命名参数无法匹配 ParamArray 参数

必须通过值传递参数数组。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 传递参数 `ByVal`，而不是命名它们。
