# <a name="bracketed-identifier-is-missing-closing-"></a>用括号标识的标识符缺少右边的“]”

转义名称中的方括号必须配对出现。  
  
 **错误 ID：** BC30034  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 在转义名称的结尾处插入右方括号。
