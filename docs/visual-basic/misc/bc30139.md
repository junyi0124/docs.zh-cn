# <a name="error-setting-assembly-manifest-option-error-message"></a>设置程序集清单选项时出错：\<error message>

Visual Basic 编译器调用程序集链接器 ( # A0 （也称为 Alink) ）以生成包含清单的程序集。 该链接器报告了查找密钥文件或查找用来对程序集签名的密钥名称时发生的错误。  
  
 **错误 ID：** BC30139  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 检查引用的错误消息并参考 [Al.exe (程序集链接器) ](../../framework/tools/al-exe-assembly-linker.md) 上列出的错误，以获取进一步的说明和建议。  
  
2. 如果仍然出现错误，则收集有关该情况的信息并通知 Microsoft 产品支持服务。
