# <a name="end-operator-expected"></a>应为“End Operator”

`End` 语句必须与它终止的块匹配。 未正确终止 `Operator` 过程定义。  
  
 **错误 ID：** BC33005  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 将 `End Operator` 语句添加到过程末尾。
