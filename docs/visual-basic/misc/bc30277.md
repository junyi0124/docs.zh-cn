# <a name="type-character-charactername-does-not-match-declared-data-type-type"></a>类型字符 " \<charactername> " 与声明的数据类型 "" 不匹配 \<type>

变量使用一种数据类型进行了声明，但是使用表示不兼容数据类型的类型字符进行了引用（例如 `K$ = 10` 声明为 `K` 时的 `Integer`）。  
  
 **错误 ID：** BC30277  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 更改变量的已声明数据类型，或更改或删除引用中的类型字符。
