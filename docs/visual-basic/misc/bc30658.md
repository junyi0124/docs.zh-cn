# <a name="property-propertyname-with-no-parameters-cannot-be-found"></a>无法找到不带参数的属性“\<propertyname>”

正在使用的特性具有一个属性，它需要一个或多个参数。  
  
 **错误 ID：** BC30658  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 为属性指定参数。
