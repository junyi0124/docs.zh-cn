# <a name="field-fieldname-is-of-an-unsupported-type"></a>字段“\<fieldname>”的类型不受支持

字段被声明为无法由 Visual Studio 表示的类型。  
  
 **错误 ID：** BC30656  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 将字段定义为支持的类型。  
  
2. 开发一种可以描述字段中数据的新类型。
