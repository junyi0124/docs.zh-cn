# <a name="unable-to-find-entry-point-name-in-dll-filename-error"></a>\<name>在 DLL "" 中找不到入口点 "" \<filename> ：\<error>

未能在指定的 DLL 中找到过程名称。  
  
 **错误 ID：** BC31014  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 验证过程名称是否拼写正确，以及大小写是否与 DLL 中函数的大小写匹配。
