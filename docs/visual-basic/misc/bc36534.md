# <a name="expression-cannot-be-converted-into-an-expression-tree"></a>无法将表达式转换为表达式树

大多数表达式可转换为表达式树，但有些无法转换。 此错误表示试图转换不能转换的表达式。  
  
 **错误 ID:** BC36534
