# <a name="statement-cannot-appear-outside-of-a-method-bodymultiline-lambda"></a>语句不能出现在方法体/多行 lambda 外。

必须在过程内使用可执行语句。  
  
 **错误 ID：** BC30689  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 将语句移动到过程或子例程中。
