# <a name="methodname-cannot-be-implemented-more-than-once"></a>" \<methodname> " 不能多次实现

你尝试实现已实现的方法。  
  
 **错误 ID：** BC30583  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 检查方法名，确保拼写正确。  
  
2. 如果该方法冗余，请删除额外的声明。
