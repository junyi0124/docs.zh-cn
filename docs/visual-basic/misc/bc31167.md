# <a name="expected--for-xml-end-tag"></a>应使用“/”作为 XML 结束标记

XML 文本的 XML 结束标记缺少必需的“/”字符。  
  
 **错误 ID：** BC31167  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 将“/”字符添加到 XML 文本的 XML 结束标记的起始位置。
