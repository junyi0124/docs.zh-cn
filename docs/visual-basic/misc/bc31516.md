# <a name="type-typename-cannot-be-used-as-an-attribute-because-it-is-not-declared-public"></a>类型“\<typename>”不可作为特性使用，因为它未声明为“Public”

特性必须是公开的。  
  
 **错误 ID：** BC31516  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 向特性类中添加 `Public` 修饰符。
