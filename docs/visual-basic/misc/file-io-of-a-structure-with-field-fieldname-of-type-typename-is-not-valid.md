# <a name="file-io-of-a-structure-with-field-fieldname-of-type-typename-is-not-valid"></a>具有类型为 "" 的字段 "" 的结构的文件 i/o \<fieldname> \<typename> 无效

试图向文件写入的结构包含不支持的类型。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 检查结构来确定不支持的类型出现的位置，然后将其更改为有效类型。
