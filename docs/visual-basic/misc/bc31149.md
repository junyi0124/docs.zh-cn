# <a name="duplicate-xml-attribute-attributename"></a>重复的 XML 特性“attributeName”

为 XML 元素文本指定的特性与之前为 XML 元素文本定义的特性匹配。 XML 元素文本中的特性名称必须是唯一的。  
  
 **错误 ID：** BC31149  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 将该特性重命名为对于 XML 元素文本唯一的名称。
