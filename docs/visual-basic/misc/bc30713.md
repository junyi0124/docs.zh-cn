# <a name="exit-statements-are-not-valid-in-the-immediate-window"></a>"Exit" 语句在即时窗口中无效

`Exit` 语句只能在源代码中使用。  
  
 **错误 ID：** BC30713  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 从调试代码中删除 `Exit` 语句。
