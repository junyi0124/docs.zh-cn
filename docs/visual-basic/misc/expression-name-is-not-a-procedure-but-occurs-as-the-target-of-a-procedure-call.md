# <a name="expression-name-is-not-a-procedure-but-occurs-as-the-target-of-a-procedure-call"></a>表达式“\<name>”不是过程，但作为过程调用的目标出现

你已使过程调用成为了一个非过程的表达式。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 请确保过程的名称拼写正确。  
  
2. 提供 <`name`> 调用的过程。
