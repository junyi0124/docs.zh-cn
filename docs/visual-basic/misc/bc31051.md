# <a name="namespace-or-type-name-has-already-been-imported"></a>已导入命名空间或类型“\<name>”

多次尝试导入命名空间。  
  
 **错误 ID：** BC31051  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 删除多余的 `Imports` 语句。
