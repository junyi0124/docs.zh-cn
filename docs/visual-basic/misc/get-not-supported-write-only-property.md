# <a name="get-not-supported-write-only-property"></a>不支持 Get（只写属性）

无法读取属性，因为它是只写的。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 检查该属性并确定可以在什么情况下对其进行设置。  
  
2. 删除对该属性的引用。
