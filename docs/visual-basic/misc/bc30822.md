# <a name="nullconstant-is-not-declared"></a>\<nullconstant>未声明 ""

未声明“\<nullconstant>”。 不再支持“Null”常量；请改用“System.DBNull”。  
  
 语句使用 `Null` 关键字，而 Visual Basic 不再支持该关键字。  
  
 **错误 ID：** BC30822  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 请使用 <xref:System.DBNull>，而不是 `Null`。 下面的示例演示这一操作。  
  
    ```vb  
    Sub TestDBNull()  
        Dim t As DataTable  
        ' Assume the DataGrid is bound to a DataTable.  
        t = CType(DataGrid1.DataSource, DataTable)  
        Dim r As DataRow  
        r = t.Rows(datagrid1.CurrentCell.RowNumber)  
        r.BeginEdit  
        r(1) = System.DBNull.Value ' Assign DBNull to the record.  
        r.EndEdit  
        r.AcceptChanges  
        If r.IsNull(1) Then  
            MsgBox("")  
        End If  
    End Sub  
    ```  
  
2. 使用对象变量时，请使用 [Nothing](../language-reference/nothing.md) 关键字进行赋值和比较。 下面的示例演示这一操作。  
  
    ```vb  
    Sub TestNothing()  
        Dim cls As Object  
        ' cls is Nothing if it has not been assigned using the New keyword.  
        If (cls Is Nothing) Then  
            MsgBox("cls is Nothing")  
        End If  
        cls = Nothing ' Assign Nothing to the class variable cls.  
    End Sub  
    ```
