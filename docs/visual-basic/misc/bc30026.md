# <a name="end-sub-expected"></a>应为“End Sub”。

`End` 语句必须与它终止的块匹配。 未正确终止 `Sub` 过程定义。  
  
 **错误 ID：** BC30026  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 将 `End Sub` 语句添加到过程末尾。
