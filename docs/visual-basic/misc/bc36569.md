# <a name="argument-not-specified-for-parameter-parametername"></a>没有为参数 "" 指定参数 \<parametername>

没有为所需形参提供实参。  
  
 **错误 ID：** BC36569  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 为指定形参提供实参。
