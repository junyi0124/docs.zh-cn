# <a name="the-file-is-already-open"></a>此文件已打开

尝试打开已打开的文件。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 在尝试打开之前关闭该文件。  
  
     \- 或 -  
  
- 从代码中删除多余的 `FileOpen` 语句。
