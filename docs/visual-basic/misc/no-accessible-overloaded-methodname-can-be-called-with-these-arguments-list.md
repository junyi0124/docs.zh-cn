# <a name="no-accessible-overloaded-methodname-can-be-called-with-these-arguments-list"></a>无法使用下列参数调用可访问的重载“\<methodname>”：\<list>

已调用重载的方法，但没有方法可以与提供的参数列表匹配。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 对照你想要调用的重载与参数列表进行匹配。  
  
- 调整参数列表，以便签名与该重载匹配。
