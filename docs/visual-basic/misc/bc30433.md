# <a name="methods-in-a-module-cannot-be-declared-specifier"></a>模块中的方法不能声明为 " \<specifier> "

你使用了在 `Module` 语句内的方法上无效的说明符。 模块永远不能实例化，不支持继承且不能实现接口。  
  
 **错误 ID：** BC30433  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 删除说明符。
