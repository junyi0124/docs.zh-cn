# <a name="stop-request-is-pending"></a>停止请求被挂起

在 Visual Studio 调试器中，表达式指定一个过程调用，但存在停止线程的请求。  
  
 调试器不会尝试在未处于活动状态的线程上调用过程。  
  
 **错误 ID：** BC30946  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 如果可能，请确定停止线程的请求的源，以便可以防止其再次出现。  
  
- 终止并重新启动调试。
