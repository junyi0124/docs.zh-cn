# <a name="type-statements-are-no-longer-supported-use-structure-statements-instead"></a>不再支持“Type”语句；请改用“Structure”语句

`Type` 关键字不能再用于定义用户定义的类型。  
  
 **错误 ID：** BC30802  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 使用 `Structure` 语句来定义新类型。
