# <a name="import-of-type-typename-from-assembly-or-module-name-failed"></a>从程序集或模块“\<name>”导入类型“\<typename>”失败

请确保所有引用库彼此兼容。  
  
 **错误 ID：** BC31091  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 验证包含引用类型的文件未损坏。
