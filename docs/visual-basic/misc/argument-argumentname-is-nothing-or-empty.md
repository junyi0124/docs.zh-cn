# <a name="argument-argumentname-is-nothing-or-empty"></a>参数“\<argumentname>”为 Nothing 或为空

参数名称无效，因为它被初始化为 `Nothing` 或为空。 参数必须具有特定的数据类型和有效的名称。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 添加有效的参数名称。
