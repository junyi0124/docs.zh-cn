# <a name="module-attribute-attributename-is-not-valid-error"></a>“\<attributename>”模块特性无效：\<error>

公共语言运行时不接受模块中的特性。

**错误 ID：** BC30130

## <a name="to-correct-this-error"></a>更正此错误

1. 从声明中删除该特性或将其替换为对上下文有效的特性。

2. 检查引用的错误信息并采取相应的操作。
