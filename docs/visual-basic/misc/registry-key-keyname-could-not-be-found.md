# <a name="registry-key-keyname-could-not-be-found"></a>找不到注册表项“\<keyname>”

找不到某个注册表项。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 检查项名的拼写。  
  
2. 确保你有足够的安全权限来访问该项。
