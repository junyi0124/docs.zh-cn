# <a name="method-methodname-is-already-declared-in-interface-interfacename"></a>\<methodname>已在接口 "" 中声明方法 "" \<interfacename>

你通过复制第二次声明了某个接口方法。  
  
 **错误 ID：** BC30377  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 删除重复的声明。
