# <a name="character-is-not-valid"></a>字符无效。

当前上下文不允许使用此字符。 例如，已声明的元素名称中不能包含波浪号 (`~`) 或竖线 (`|`)。  
  
 **错误 ID：** BC30037  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 检查是否在编程元素中意外插入了额外字符。  
  
2. 检查在拼写或构造编程元素时是否只使用了对这种元素有效的字符。
