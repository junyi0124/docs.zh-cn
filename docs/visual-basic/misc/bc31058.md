# <a name="conflicts-with-name1-which-is-implicitly-created-for-name2-in-type-name3"></a>与为 "" \<name1> 中的 "" 隐式创建 \<name2> 的 \<type> \<name3> "" 冲突

类型成员的名称与另一个隐式创建的成员相冲突。  
  
 **错误 ID：** BC31058  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 重命名显式声明的成员，以消除命名冲突。
