# <a name="argument-value-pathname-contains-characters-that-are-not-valid-in-a-path-name"></a>参数值“\<pathname>”包含在路径名中无效的字符

路径名包含一个或多个无效字符。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 从路径名中删除无效字符。
