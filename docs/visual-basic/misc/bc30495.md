# <a name="exponent-is-not-valid"></a>指数无效

试图使用的指数与浮点数同时使用无效。 通常，当浮点数类型与保存浮点数所需的大小不一致时，将导致此错误。  
  
 **错误 ID:** BC30495  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 检查浮点数的类型，并将其更改为有效的类型。
