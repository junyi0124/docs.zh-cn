# <a name="qualifiedmethod-is-not-accessible-in-this-context-because-it-is-modifier"></a>"" \<qualifiedmethod> 是 ""，因此它在此上下文中不可访问 \<modifier>

尝试访问一个由于应用了修饰符而在此上下文中不可访问的方法。  
  
 **错误 ID：** BC30390  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 删除此修饰符。
